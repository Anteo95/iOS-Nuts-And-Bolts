## Contributing

Before contrubiting any peace of code to this repo, plesease go over this checklist:
* Clean up the code
    * Remove any refrence to the project that peace of code was created in
    * Reduce the dependencies to minimum
* In folder's README.md reference all dependencis that have left, e.g. RxSwift
* Document all of the functions
* Write a peace of text describing what that code does in folder's README.md file and / or provide a screanshot if possible
